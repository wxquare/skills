# Test Prompts

用于验证 `integration-test-result` 是否会被正确触发的典型请求。

1. `把这轮集成测试执行结果整理成结构化报告。`
2. `Summarize these failed cases, logs, and traces into a reviewable test result report.`
3. `请按 Pass / Fail / Blocked / Not Run 汇总这次测试结果。`
4. `I need a release-readiness summary from this manual test execution evidence.`
5. `根据这些日志和执行备注，帮我出测试结果结论和风险评估。`
6. `整理这次 CI 集成测试失败情况，给出下一步建议。`
