# Test Prompts

用于验证 `integration-test-plan` 是否会被正确触发的典型请求。

1. `基于这个 PRD、TD 和 MR，帮我出一份集成测试方案。`
2. `Generate a reviewable integration test plan for these API changes.`
3. `请整理测试范围、风险点和 case proposal，不要直接写测试代码。`
4. `I need a test plan report before manual execution for this service integration change.`
5. `把这次接口改动整理成可 review 的测试计划。`
6. `根据 diff 和 schema 设计本轮必须测/建议测/不测项。`
