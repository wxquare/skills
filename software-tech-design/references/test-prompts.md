# Test Prompts

用于验证 `technical-design` 是否会被正确触发的典型请求。

1. `帮我写一份这个需求的 TD。`
2. `Review this architecture proposal and point out rollout and rollback gaps.`
3. `把这份会议纪要整理成技术方案，重点写关键决策。`
4. `请审一下这份设计文档，看看风险评审和 fallback 是否完整。`
5. `Write a technical design for this migration plan with explicit assumptions.`
6. `这份架构方案可以过评审吗？`
