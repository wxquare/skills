# Test Prompts

用于验证 `code-review` 是否会被正确触发的典型请求。

1. `帮我 review 这个 PR，重点看是否 ready to merge。`
2. `请审查这组 diff，找 correctness risk 和 test gap。`
3. `这个重构可以合并吗？请按严重度列 findings。`
4. `Review this patch for maintainability and rollback risk.`
5. `检查这次支付链路改动有没有上线风险。`
6. `Can you review these changes and focus on concurrency and idempotency issues?`
